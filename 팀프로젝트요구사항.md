팀 프로젝트 구현 요구사항
• vite 로 생성한 Typescript 기반의 React 프로젝트
• Node 버전 22 이상
• React 버전 19, 함수형 컴포넌트와 훅 사용
• Tailwindcss 버전 4
• Tailwindcss 이용한 반응형 디자인을 구현
• 추가 CSS 또는 추가 CSS Library 사용 가능
• 반응형 디자인은 tailwindcss 브레이크포인트(sm, md, lg 등)만 인정
• react-router 를 활용하여 3페이지 이상의 화면 구현
• 공공데이터 API(https://www.data.go.kr/) 1개 이상 활용
• 백엔드 - Firebase 사용. 앱 기준 백엔드를 활용한 CRUD 기능 각 1개 이상
• 다국어 적용 - 한국어 + 추가 언어 1개 이상 (단, 데이터베이스에 저장된 데이터는 번역하지 않음)
• UI 스타일 가이드 작성 및 준수 - 프로젝트안에 readme.txt 로 제출
• 컬러 팔렛트, 폰트, 버튼, 입력필드 스타일 등
• 모든 UI 에서 로딩, 성공, 오류 등에 대한 적절한 안내가 이루어 질 것
